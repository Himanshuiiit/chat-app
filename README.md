<h1> Chit-Chat</h1>
<u><h3>The chat app</h3></u>

<p>
  A chating app made using react, node, express and mongodb.<br>
  Chakra UI is used to create the components and socket.io is used for real time reaction and animation in the app.<br>
  cloudinary is used to store the images.
</p>
<ul>
  <h4>Features of the app</h4>
  <li>Sign up and login</li>
  <li>Group or one to one chat</li>
  <li>Admin access to the groups</li>
  <li>Encryption in database</li>
</ul>
